What is RoboCSE?


The Robot Common-Sense Embedding (RoboCSE) aims at providing robots with a representation of "common-sense" household knowledge.

How does RoboCSE work?


RoboCSE represents a knowledge graph containing household domain knowledge as an embedding. Please view the following video with audio to learn more.
